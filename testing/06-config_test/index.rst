配置格式
""""""""""""

Razor支持YAML 和 JSON 两种的配置文件，暂时不支持默认配置，每项配置都需要填写。

参数示例
""""""""""""

:server:
    - host: 127.0.0.1
    - port: 8080

:database:
    - host: ued.hex2dec.com
    - post: 3306
    - username: razor
    - password: razor
    - db: razor_lcs

参数描述
""""""""""""

服务器（server）
*******************

====== ============ ===================
配置项  含义         建议值
====== ============ ===================
host   服务监听地址  业务IP地址
port   服务监听端口  HTTP 80 / HTTPS 443
====== ============ ===================

数据库（database）
*******************

========= ============== ===========
配置项     含义           建议值
========= ============== ===========
host      数据库服务地址  127.0.0.1
port      数据库服务端口  3306
username  数据库用户名    razor
password  数据库密码      razor
db        数据库库名称    razor
========= ============== ===========
