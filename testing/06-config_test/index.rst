配置格式
""""""""""""

Razor支持YAML 和 JSON 两种的配置文件，暂时不支持默认配置，每项配置都需要填写。

参数示例
""""""""""""

:server:
    - host: 127.0.0.1
    - port: 8080
  
:database:
    - host: ued.hex2dec.com
    - post: 3306
    - username: razor
    - password: razor
    - db: razor_lcs

:redis:
    - url: 127.0.0.1:6379

:register:
    - auto_accept: false
    - admin_email: contact@unitedstack.com

:ui:
    - header:
        - logo: UNKNOW
    - auth:
        - logo: UNKNOW
    - copyright:
        - name: 同方有云（北京）科技有限公司
    - about:
        - company_name: 同方有云（北京）科技有限公司
        - company_address: 北京市海定区王庄路1号清华同方科技大厦D座6层
        - phone: 400-898-5401
        - email: contact@unitedstack.com
        - logo: /logo_vertical.png
    - docs:
        - user_guide: UNKNOW
        - admin_guide: UNKNOW

参数描述
""""""""""""

服务器（server）
*******************

====== ============ ===================
配置项  含义         建议值
====== ============ ===================
host   服务监听地址  业务IP地址
port   服务监听端口  HTTP 80 / HTTPS 443
====== ============ ===================

数据库（database）
*******************

========= ============== ===========
配置项     含义           建议值
========= ============== ===========
host      数据库服务地址  127.0.0.1
port      数据库服务端口  3306
username  数据库用户名    razor
password  数据库密码      razor
db        数据库库名称    razor
========= ============== ===========

缓存（redis）
***************
======= ============================ ===========
配置项   含义                         建议值
======= ============================ ===========
url     redis 服务 URL，
        参考下述描述   
======= ============================ ===========

**Redis URL 配置方法**

Redis 实例：

redis://<user>:<pass>@localhost:6379/<db>

redis:// 127.0.0.1:6379

其中db为redis数据库编号，
例如配置数据库编号为1的redis数据库：redis:// 127.0.0.1:6379/1

注册（register）
*****************

============= ================================ ========
配置项         含义                             建议值
============= ================================ ========
auto_accept   自动注册审批，true 表示自动审批.   false
              false 表示管理员手工审批
admin_email   管理员审批时，发送通知的邮件地址    
============= ================================ ========
