# 并发读取文件夹的信息

## 问题描述：

实现一个程序计算并打印输入的目录下所有文件的总数和总大小(以GB计算)。完成之后你将熟悉select、WaitGroup、ioutil的用法

## 要点：

* 并发读取文件(夹)信息。
* 限制开启的goroutines的最大数量。
* 运行时每隔500ms打印当前已经统计的文件数和总大小（使用命令行参数指定此功能是否启用）。

## 拓展

在执行中在有外部输入时退出程序。
